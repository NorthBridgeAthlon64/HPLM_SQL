### P-01 登录/注册页

![登录页](images/p01-login.png)

<!-- 替换为你的实际截图路径 -->

界面说明：系统入口页面，提供用户登录和注册功能。用户输入用户名和密码进行身份验证，系统根据角色（管理员/普通员工）跳转到不同主页。新用户可点击注册链接创建账号，注册时需要填写基本信息。

### P-02 仪表盘/首页

![界面截图](images/p02-xxx.png)

界面说明：系统主控台，展示关键业务指标和待办事项。包括：低库存预警元器件列表（库存低于min_stock阈值）、待处理维修工单数量、最近生产批次状态统计、近期客户评价趋势图。各卡片可点击跳转到对应管理页面。

### P-03 元器件管理页

![界面截图](images/p03-xxx.png)

界面说明：元器件信息的增删改查页面。支持按元器件编码、名称、供应商筛选；展示元器件列表（含当前库存、当前价格、最低库存阈值）；提供新增/编辑元器件表单（含规格、单位、供应商、当前价格等字段）。点击库存数字可查看该元器件的库存流水明细。

### P-04 产品版本与BOM配置页

![界面截图](images/p04-xxx.png)

界面说明：产品版本管理及BOM配置页面。左侧展示产品列表，右侧展示选中产品的所有版本（含版本号、标题、状态、发布时间）。点击版本进入BOM配置界面，可查看该版本的BOM清单，支持添加/修改/删除元器件用量和PCB位置。同时显示该版本的预估单台成本和建议售价。

### P-05 生产批次管理与测试页

![界面截图](images/p05-xxx.png)

界面说明：生产批次创建和测试记录管理页面。支持创建新生产批次（选择产品版本、输入批次号、投产数量）；展示已有批次列表（批次号、版本、质量状态、生产日期）；点击批次可查看投料明细（系统根据BOM自动生成）和测试记录；测试工程师可为批次添加测试结果（通过/失败），记录测试数据和测试项目。

### P-06 维修工单管理页

![界面截图](images/p06-xxx.png)

界面说明：售后维修工单管理页面。支持创建维修工单（选择客户、产品版本、填写故障描述）；展示工单列表（工单号、客户、版本、接修日期、状态）；点击工单进入详情页，可添加维修换料记录（选择更换的元器件、数量、单价），系统自动计算维修费用；支持更新工单状态和完成日期。

### P-07 故障溯源分析页

![界面截图](images/p07-xxx.png)

界面说明：产品故障溯源分析页面，是本系统的核心价值功能。用户输入产品编号或维修单号，系统展示完整的溯源链路图：产品 → 所属生产批次 → 该批次使用的元器件列表（含用量、用料时单价）→ 每个元器件的供应商信息 → 同批次其他产品的测试记录。支持导出溯源报告。

### P-08 客户评价与反馈页

![界面截图](images/p08-xxx.png)

界面说明：客户产品评价管理页面。客户登录后可查看已购产品版本列表，对每个版本进行评分和文字评价；管理员/产品经理可查看各版本的平均评分和评价明细，支持筛选低评分版本（如<3星）进行质量问题追踪。评价数据与具体产品版本绑定，确保版本间评价不混淆。


## 界面-数据库交互映射表

| 界面编号 | 界面名称 | 用户操作 | 数据库操作类型 | 涉及的表 | 关联查询/主要SQL描述（必填） |
|---------|---------|---------|--------------|---------|---------------------------|
| P-01 | 登录页 | 输入工号和密码登录 | SELECT | Employee | SELECT employee_id, name, position FROM Employee WHERE employee_no = ? AND password = ? |
| P-01 | 登录页 | 新用户注册 | INSERT | Employee | INSERT INTO Employee (employee_no, name, password, position, phone, email) VALUES (?, ?, ?, ?, ?, ?) |
| P-02 | 仪表盘 | 查看低库存预警 | SELECT | Component, Inventory | SELECT c.component_code, c.name, c.min_stock, i.quantity FROM Component c JOIN Inventory i ON c.component_id = i.component_id WHERE i.quantity < c.min_stock |
| P-02 | 仪表盘 | 查看待处理维修工单数量 | SELECT | RepairRecord | SELECT COUNT(*) FROM RepairRecord WHERE repair_status IN ('received', 'in_progress') |
| P-02 | 仪表盘 | 查看最近批次质量统计 | SELECT | ProductionBatch | SELECT quality_status, COUNT(*) FROM ProductionBatch WHERE production_date >= CURRENT_DATE - INTERVAL '30 days' GROUP BY quality_status |
| P-03 | 元器件管理页 | 查询元器件列表 | SELECT | Component, Supplier, Inventory | SELECT c.*, s.name AS supplier_name, i.quantity, i.avg_cost FROM Component c LEFT JOIN Supplier s ON c.supplier_id = s.supplier_id LEFT JOIN Inventory i ON c.component_id = i.component_id WHERE c.component_code LIKE ? OR c.name LIKE ? |
| P-03 | 元器件管理页 | 新增元器件 | INSERT | Component | INSERT INTO Component (component_code, name, specification, unit, min_stock, current_price, supplier_id) VALUES (?, ?, ?, ?, ?, ?, ?) |
| P-03 | 元器件管理页 | 编辑元器件信息 | UPDATE | Component | UPDATE Component SET name=?, specification=?, unit=?, min_stock=?, current_price=?, supplier_id=? WHERE component_id=? |
| P-03 | 元器件管理页 | 查看元器件库存流水 | SELECT | InventoryTransaction | SELECT transaction_no, transaction_type, quantity, unit_price, transaction_date, reference_no FROM InventoryTransaction WHERE component_id=? ORDER BY transaction_date DESC |
| P-04 | 产品版本与BOM配置页 | 查询产品列表 | SELECT | Product | SELECT product_id, product_code, name FROM Product ORDER BY product_id |
| P-04 | 产品版本与BOM配置页 | 查询产品版本列表 | SELECT | ProductVersion, Employee | SELECT pv.*, e.name AS release_engineer FROM ProductVersion pv LEFT JOIN Employee e ON pv.release_engineer_id = e.employee_id WHERE pv.product_id = ? ORDER BY pv.version_number DESC |
| P-04 | 产品版本与BOM配置页 | 创建新产品版本 | INSERT | ProductVersion | INSERT INTO ProductVersion (product_id, version_number, version_title, version_note, status, release_engineer_id) VALUES (?, ?, ?, ?, ?, ?) |
| P-04 | 产品版本与BOM配置页 | 查看版本BOM清单 | SELECT | ProductBOM, Component | SELECT pb.bom_id, c.component_code, c.name, c.specification, pb.quantity, pb.position FROM ProductBOM pb JOIN Component c ON pb.component_id = c.component_id WHERE pb.version_id = ? |
| P-04 | 产品版本与BOM配置页 | 向BOM添加元器件 | INSERT | ProductBOM | INSERT INTO ProductBOM (version_id, component_id, quantity, position) VALUES (?, ?, ?, ?) |
| P-04 | 产品版本与BOM配置页 | 修改BOM用量 | UPDATE | ProductBOM | UPDATE ProductBOM SET quantity=?, position=? WHERE bom_id=? |
| P-04 | 产品版本与BOM配置页 | 从BOM删除元器件 | DELETE | ProductBOM | DELETE FROM ProductBOM WHERE bom_id=? |
| P-05 | 生产批次管理页 | 创建生产批次 | INSERT | ProductionBatch | INSERT INTO ProductionBatch (batch_no, version_id, quantity, production_date, tester_id, quality_status) VALUES (?, ?, ?, ?, ?, 'pending') |
| P-05 | 生产批次管理页 | 查询批次列表 | SELECT | ProductionBatch, ProductVersion, Product | SELECT pb.*, pv.version_number, p.name AS product_name FROM ProductionBatch pb JOIN ProductVersion pv ON pb.version_id = pv.version_id JOIN Product p ON pv.product_id = p.product_id ORDER BY pb.production_date DESC |
| P-05 | 生产批次管理页 | 查看批次投料明细 | SELECT | BatchMaterial, Component | SELECT bm.*, c.component_code, c.name, c.unit FROM BatchMaterial bm JOIN Component c ON bm.component_id = c.component_id WHERE bm.batch_id = ? |
| P-05 | 生产批次管理页 | 生产投料时扣减库存 | INSERT/UPDATE | BatchMaterial, Inventory, InventoryTransaction | 1. INSERT INTO BatchMaterial (batch_id, component_id, used_quantity, unit_cost_at_time) VALUES (?, ?, ?, ?); 2. UPDATE Inventory SET quantity = quantity - ? WHERE component_id = ? AND warehouse_id = ?; 3. INSERT INTO InventoryTransaction (transaction_no, component_id, warehouse_id, transaction_type, quantity, unit_price, total_price, reference_no, operator_id) VALUES (?, ?, ?, 'out', ?, ?, ?, ?, ?) |
| P-05 | 生产批次管理页 | 添加测试记录 | INSERT | TestRecord | INSERT INTO TestRecord (batch_id, tester_id, test_date, test_item, test_result, test_data) VALUES (?, ?, ?, ?, ?, ?) |
| P-05 | 生产批次管理页 | 更新批次质量状态 | UPDATE | ProductionBatch | UPDATE ProductionBatch SET quality_status = ?, tester_id = ? WHERE batch_id = ? |
| P-06 | 维修工单管理页 | 查询客户列表 | SELECT | Customer | SELECT customer_id, customer_code, name, phone FROM Customer WHERE name LIKE ? OR customer_code LIKE ? |
| P-06 | 维修工单管理页 | 创建维修工单 | INSERT | RepairRecord | INSERT INTO RepairRecord (repair_no, version_id, batch_id, customer_id, receive_date, fault_description, repair_status, repair_cost) VALUES (?, ?, ?, ?, ?, ?, 'received', 0) |
| P-06 | 维修工单管理页 | 查询维修工单列表 | SELECT | RepairRecord, Customer, ProductVersion | SELECT r.*, c.name AS customer_name, pv.version_number FROM RepairRecord r JOIN Customer c ON r.customer_id = c.customer_id JOIN ProductVersion pv ON r.version_id = pv.version_id ORDER BY r.receive_date DESC |
| P-06 | 维修工单管理页 | 添加维修换料 | INSERT/UPDATE | RepairMaterial, Inventory, InventoryTransaction, RepairRecord | 1. INSERT INTO RepairMaterial (repair_id, component_id, quantity, unit_cost_at_time) VALUES (?, ?, ?, ?); 2. UPDATE Inventory SET quantity = quantity - ? WHERE component_id = ?; 3. INSERT INTO InventoryTransaction (...) VALUES (...); 4. UPDATE RepairRecord SET repair_cost = repair_cost + ? WHERE repair_id = ? |
| P-06 | 维修工单管理页 | 更新工单状态 | UPDATE | RepairRecord | UPDATE RepairRecord SET repair_status = ?, complete_date = ? WHERE repair_id = ? |
| P-07 | 故障溯源分析页 | 根据维修单号溯源 | SELECT | RepairRecord, ProductionBatch, BatchMaterial, Component, Supplier | SELECT r.repair_no, pv.version_number, pb.batch_no, c.component_code, c.name, bm.used_quantity, bm.unit_cost_at_time, s.name AS supplier_name FROM RepairRecord r LEFT JOIN ProductVersion pv ON r.version_id = pv.version_id LEFT JOIN ProductionBatch pb ON r.batch_id = pb.batch_id LEFT JOIN BatchMaterial bm ON pb.batch_id = bm.batch_id LEFT JOIN Component c ON bm.component_id = c.component_id LEFT JOIN Supplier s ON c.supplier_id = s.supplier_id WHERE r.repair_no = ? |
| P-07 | 故障溯源分析页 | 查询同批次其他产品的测试记录 | SELECT | RepairRecord, ProductionBatch, TestRecord | SELECT t.test_id, t.test_item, t.test_result, t.test_data, t.test_date FROM RepairRecord r JOIN ProductionBatch pb ON r.batch_id = pb.batch_id JOIN TestRecord t ON pb.batch_id = t.batch_id WHERE r.repair_no = ? |
| P-07 | 故障溯源分析页 | 导出溯源报告 | SELECT | 多表联合查询 | 组合上述SQL查询结果，生成完整报告。核心溯源SQL见上行 |
| P-08 | 客户评价页 | 查询客户可评价的产品版本 | SELECT | RepairRecord, ProductVersion | SELECT DISTINCT pv.version_id, pv.version_number, pv.version_title FROM RepairRecord r JOIN ProductVersion pv ON r.version_id = pv.version_id WHERE r.customer_id = ? AND r.repair_status = 'completed' |
| P-08 | 客户评价页 | 提交产品评价 | INSERT | ProductFeedback | INSERT INTO ProductFeedback (version_id, customer_id, rating, comment, feedback_date, status) VALUES (?, ?, ?, ?, CURRENT_DATE, 'pending') |
| P-08 | 客户评价页 | 查看版本评价列表（客户端） | SELECT | ProductFeedback | SELECT rating, comment, feedback_date FROM ProductFeedback WHERE version_id = ? AND status = 'approved' ORDER BY feedback_date DESC |
| P-08 | 客户评价页 | 查看版本评分统计（管理端） | SELECT | ProductFeedback, ProductVersion | SELECT pv.version_number, AVG(pf.rating) AS avg_rating, COUNT(pf.feedback_id) AS total_feedbacks FROM ProductVersion pv LEFT JOIN ProductFeedback pf ON pv.version_id = pf.version_id AND pf.status = 'approved' WHERE pv.product_id = ? GROUP BY pv.version_id ORDER BY pv.version_number DESC |
| P-08 | 客户评价页 | 审核评价（管理员） | UPDATE | ProductFeedback | UPDATE ProductFeedback SET status = ? WHERE feedback_id = ? |
| P-08 | 客户评价页 | 查询低评分版本（<3星） | SELECT | ProductVersion, ProductFeedback | SELECT pv.version_number, pv.version_title, AVG(pf.rating) AS avg_rating FROM ProductVersion pv JOIN ProductFeedback pf ON pv.version_id = pf.version_id WHERE pf.status = 'approved' GROUP BY pv.version_id HAVING AVG(pf.rating) < 3 |
