# 模板文件使用说明

## 文件列表

本模板包包含以下文件：

| 文件 | 说明 |
|------|------|
| 01_功能说明.md | 功能说明文档模板 |
| 02_数据库设计.md | 数据库设计文档模板 |
| 03_功能设计.md | 功能设计文档模板 |
| 04_AI使用记录.md | AI使用记录模板 |
| images/ | 图片存放文件夹（界面截图等） |

## 使用步骤

1. 解压本模板包
2. 用任意 Markdown 编辑器（推荐 Typora、VS Code）打开各 .md 文件
3. 按照模板中的提示和格式要求填写内容
4. 将截图等图片文件放入 images/ 文件夹
5. 完成后，将所有文件（含 images 文件夹）打包为 zip 压缩包

## 注意事项

- **不得更改任何文件名称**，包括 .md 文件和 images 文件夹
- **不得更改文件格式**，所有文档必须为 .md（Markdown）格式
- **不得修改模板中的表格表头**
- **不得在压缩包内嵌套子目录**，所有文件放在压缩包根目录
- 压缩包命名格式：`DB_Proj_学号_姓名_作品名称.zip`
- 05_系统录屏.mp4 由你自己录制并添加到压缩包中（本模板不含此文件）
